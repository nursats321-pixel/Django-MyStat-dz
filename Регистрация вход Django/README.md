Сделать так, чтобы при запуске проекта нас встречала не главная страница чата, а страница авторизации/регистрации. Пока не залогинишься - в чат не попадёшь

Что нужно сделать:
Главная страница теперь - это Логин (/)

Перенастройте urls.py так, чтобы по пустому адресу '' открывалась страница входа (login.html)

На странице входа сделать форму (Логин + Пароль) и кнопку "Войти"

Снизу под формой сделать ссылку: "Ещё нет аккаунта? Зарегистрироваться". Она должна вести на /register/

Страница регистрации (/register/)

Создать шаблон register.html с формой (Логин + Пароль) и кнопкой "Создать аккаунт"

При отправке формы юзер должен создаваться в базе данных через:
User.objects.create_user(username=..., password=...)

После успешной регистрации сайт должен автоматически кидать юзера обратно на страницу логина

Контроль через Админку

Зайдите в /admin в раздел Users

Вы должны видеть список всех, кто зарегался через вашу форму, и у вас должна быть возможность удалить любого юзера одной кнопкой прямо из таблицы

Как это работает в итоге
Запускаем сайт -> Видим логин -> Жмём "Регистрация" -> Создаём аккаунт -> Нас кидает на логин -> Вводим данные -> Попадаем в наш чат
